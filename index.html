Asosiy texnologiyalar

HTML - Hypertext Markup Language, gipermatnli markerlash tili. Veb sahifalarni yasash uchun ishlatiladigan standart markerlash tili, dasturlash tili emas. Boshqacha qilib aytadigan bo'lsak, bu til veb sahifaning skeleti hisoblanadi. Muallifi - Tim Berners Li.

CSS - Cascading Style Sheets, veb sahifaga stil berish tili. CSS veb sahifadagi elementlarning qanday ko'rinishini - rangini, shriftini, o'lchamlarini, qayerda turishini aniqlash uchun foydalaniladigan til. CSS veb sahifamizni chiroyli qilishda bizga juda katta yordam bera oladi.

JavaScript - qisqartmasi JS, ECMAScript spetsifikatsiyasiga mos keladigan, 10 kunda ishlab chiqilgan dasturlash tili. JavaScript yuqori darajadagi, just-in-time kompilyatsiya bo'ladigan, multi-paradigmali til. HTML va CSS bilan bir qatorda WWWning asosiy texnologiyalaridan biri. 1995-yil 4-dekabrda keng ommaga taqdim etilgan. Muallifi - Brendn Eych.

"Hamma dasturlashni o'rganishi kerak. Chunki, u fikrlashni o'rgatadi."

Stiv Jobs, Apple asoschisi

Boshqa texnologiyalar

Bootstrap

Bootstrap - bepul, ochiq kodli, moslashuvchan, mobile-first veb-saytlarni qurish uchun mo'ljallangan, BootstrapCDN li dunyoning eng mashhur front-end freymvorki. 2011-yil 19-avgustda chiqqan. HTML, CSS, Less, Sass va JavaScript da yozilgan. Mualliflari - Mark Otto va Yakob Tornton. MIT litsenziyasiga ega.

MongoDB

MongoDB - JavaScript, Node.js bilan yaxshi chiqishadi. Zamonaviy ilovalar uchun ma'lumotlar bazasi. MongoDB umumiy maqsadlarda foydalaniladigan, hujjatga asoslangan, bulutli era va ko'plab ilova dasturchilar uchun qurilgan, distributlashgan ma'lumotlar bazasi.

Express.js

Express.js - Node.js uchun tezkor, moslashuvchan, minimalistik veb-freymvork. Bunda ko'plab HTTP metodlar, oraliq qayta ishlovchi(middleware)lar yordamida, ishonchli tezkor va oson API server yaratish mumkin bo'ladi.

React.js

React.js - foydalanuvchi interfeyslari yoki UI komponentalarini qurish uchun foydalaniladigan ochiq kodli, front-end, JavaScript kutubxonasi. Muallifi - Jordan Walke. 2013-yil 29-mayda chiqqan. MIT litsenziyasiga ega.

Node.js

Node.js - JavaScriptni brauzerdan tashqarida ishlatib beruvchi muhit. Nodeda Express.js freymvorki yordamida API lar qurish imkoni yaratilgan. Back-endda keng qo'llaniladi. Hozirgi kunda rivojlanib bormoqda. PHP ning asosiy raqobatchisi.
